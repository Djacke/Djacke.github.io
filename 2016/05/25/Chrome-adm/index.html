chrome启用第三方插件

发表于 2016-05-25   |   分类于 Hexo ， Chrome plugin
  1. 首先下载一个chrome的模板文件备用，下载地址：
    http://pan.baidu.com/s/1pJJSzHL

  1. 打开window运行框输入gpedit.msc 回车，如下图：

  2. 打开本地策略编辑器，右键模板管理，然后在弹出框点添加按钮，选择第一步下载的chrome.adm文件

  3. 添加完点击关闭，然后再点击管理模板 ，看子菜单是否有经典管理模板（ADM）节点，继续查看子菜单Goodgle->Google Chrome->扩展程序，

  4. 双击点开配置扩展程序安装白名单 ，选中已启用 ，点击显示，添加第三方插件ID，点击确定

  5. 打开chrome的选项菜单，选择更多工具 -> 扩展程序 ，找到需要启用的第三方插件，选择已启用，重启浏览器OK。
